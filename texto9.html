|                                                                        |
| Han pasado casi ocho meses, cuatro veces nos hemos visto en doscientos |
| cuarenta días. A veces siento que nunca te podré olvidar. Es complicado|
| porque te he superado, pero sigo aferrada a lo que no fue, a lo que    |
| construimos sobre el aire y desapareció en la lluvia.                  |
|                                                                        |
| Tengo miedo, tengo miedo de no poder volver a sentir nada, porque      |
| ahora conozco, hablo, me abro, pero es nada, porque mi confianza y     |
| bondad te las llevaste. Han pasado doscientos cuarenta días y sigo sin |
| poder sentir nada. Estoy mejor, estoy feliz, pero no tengo la          |
| capacidad de sentir la chispa que contigo creció en mí como una        |
| absoluta torre de electricidad. Tal vez me quemé o tal vez nunca       |
| existió. Tal vez nunca me lograste amar como yo te amé, y eso es lo    |
| que más me dolió.                                                      |
|                                                                        |
| ¿Por qué nunca pudiste darme lo mejor de ti, cuando yo te entregué mi  |
| cuerpo y alma en una cajita de cartón con chocolates de souvenir?      |
|                                                                        |
| A veces siento que herviste mi corazón y la sangre se esparció por toda|
| la cocina; se hizo un desastre medio poético que limpiaste con ácido y |
| lo guardaste para el recuerdo. Cuando intentaste volver a construirlo, |
| estaba tan desmoronado que se hizo pedazos. Entre el olor evaporado    |
| de mi órgano, te asfixiabas. Yo también te hice daño, tal vez te       |
| envenené. En mi defensa, era la manera de la carne de gritar que se    |
| está cociendo, que necesita vigilancia. Pero no vigilaste, te          |
| acomodaste entre el aire tóxico y pesado, te acurrucaste y dormiste    |
| mientras saboreabas mis huesos para tu placer, me comiste desde la     |
| boca hasta mis entrañas, te empachaste de mi sangre.                   |
|                                                                        |
| ¿Qué tanto te gustó alimentarte de mí?                                 |
| Si el amor es una metáfora del canibalismo, ¿cuántas veces más me      |
| podrías comer?                                                         |
